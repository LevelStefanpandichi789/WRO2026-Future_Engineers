# 3D-models - 3D Models

This folder contains 3D design files for the robot's components. These files include parts used in the vehicle's steering and drivebase. Formats include STL and/or STEP files.

### Models

- `RCcar_1.stl`: Front wheel with integrated bearing support, designed for smooth steering and reduced wobbling during high-speed movement.
- `RCcar_2.stl`: Rear drive wheel optimized for traction and compatibility with the custom differential system.
- `RCcar_3.stl`: Left steering hub connecting the front wheel to the steering linkage while allowing low-friction rotation.
- `RCcar_4.stl`: Right steering hub designed symmetrically to the left hub for stable and precise steering control.
- `RCcar_5.stl`: Gear mounted on the drive motor shaft, transferring power to the differential input gear.
- `RCcar_6.stl`: Main rear drivetrain support holding the motor, differential, wheel axles, and bearings in a compact assembly.
- `RCcar_7.stl`: Structural support used to mount the impeller motor and maintain stable airflow underneath the robot.
- `RCcar_8.stl`: Lightweight impeller fan designed to generate aerodynamic downforce for increased traction.

### Archive
- `old-3D-models/`: Contains earlier versions of the robot's 3D models for reference.

## Usage
- Download the `.stl` or `.step`(if available) files to fabricate the components using a 3D printer or CNC machine.
- Ensure the parts are assembled according to the robot's full assembled design.
